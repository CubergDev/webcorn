Earth assets used by the Webcorn background scene.

- `earth-blue-marble-4096.jpg` is an optimized 4K local copy of NASA Blue Marble: Next Generation January base map: https://science.nasa.gov/earth/earth-observatory/blue-marble-next-generation/base-map/
- `earth-city-lights-4096.jpg` is an optimized 4K local copy of NASA Black Marble 2016 night lights: https://science.nasa.gov/earth/earth-observatory/earth-at-night/maps/
- `earth-clouds-1024.png`, `earth-specular-2048.jpg`, and `earth-normal-2048.jpg` are Three.js example planet textures used for cloud, ocean-highlight, and surface-relief layers: https://threejs.org/examples/textures/planets/
- `moon-1024.jpg` is a Three.js example Moon texture used for the background Moon object: https://threejs.org/examples/textures/planets/

The files are stored locally so the landing page does not depend on external image requests during load.
